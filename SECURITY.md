## 报告安全漏洞

如果你发现安全漏洞，请通过以下方式报告：

**请勿在 GitHub Issue 中报告安全漏洞**

请发送邮件至项目维护者或通过私人消息联系，我们会：

1. 确认收到报告
2. 告诉你我们计划如何修复
3. 发布修复方案并公开致谢

## 安全更新

一旦报告的漏洞被确认并修复，我们将在最短时间内发布安全更新。
